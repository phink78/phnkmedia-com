# Security Policy

## Project Overview

This repository hosts the PHNK Media and Design public website.

- Domain: phnkmedia.com
- Primary branch: latest default branch
- Privacy details: see privacy.html

## Supported Versions

Security fixes are applied to the latest code on the default branch.
Older snapshots, backups, and deleted files are not supported.

## How to Report a Vulnerability

Please report vulnerabilities privately by email.

- Contact: patrick@phnkmedia.com
- Subject: Security Report - phnkmedia.com

Please include as much of the following as possible:

- Summary of the issue
- Affected URL or file path
- Reproduction steps
- Expected vs actual behavior
- Impact assessment
- Proof of concept (screenshots, request samples, or minimal script)
- Suggested fix (optional)

## Response Timeline Targets

- Acknowledgement: within 3 business days
- Initial triage: within 7 business days
- Status updates: provided for valid reports until closure
- Remediation timeline: based on severity, complexity, and operational risk

These are targets, not guarantees.

## Scope

In scope:

- Source code in this repository
- Pages deployed on phnkmedia.com
- Consent and analytics behavior implemented by this site

Out of scope:

- Third-party platforms and infrastructure not controlled by this repository
- Social media platforms and external websites linked from this site
- Issues that require physical access to user devices
- Denial-of-service, stress testing, or traffic flooding
- Non-security UI or content issues

## Testing Rules

Please test responsibly.

- Do not degrade availability or performance
- Do not access, alter, or destroy data you do not own
- Do not attempt account takeover or privilege abuse beyond proof of concept
- Use the minimum data and requests needed to demonstrate impact

## Coordinated Disclosure

Please allow reasonable time to investigate and remediate before public disclosure.
If you plan to publish findings, coordinate a disclosure date first.

## Rewards

This project does not currently offer a paid bug bounty program.

Thank you for helping keep this project secure.
